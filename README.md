# ケンとハナの食卓バトル

料理をテーマにした AI 制作ラジオ番組プロジェクト。

毎回1つの料理を取り上げ、栄養素・歴史・文化的背景をもとに友人2人（ケンとハナ）が口論する料理雑談ラジオ。
Claude Code が台本生成・音声合成・GitHub push までの全工程を自動実行する。

## 技術構成

- **台本生成**：Claude Code（Anthropic）
- **音声合成**：VOICEVOX（ローカル起動）
- **バージョン管理**：GitHub

## 制作フロー

```
テーマ受取
  ↓
ウェブリサーチ（Claude Code の WebSearch ツール）
  ↓
台本生成（キャラ設定に従い自動生成）
  ↓
レビュー（基準を満たすまで自動修正）
  ↓
音声生成（VOICEVOX / tts.py）
  ↓
GitHub push（台本・音声・ナレッジのみ）
```
