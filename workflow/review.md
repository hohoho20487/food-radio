# チェック手順

## 台本生成後に必ず実行すること

### チェックリスト
- [ ] ケンの口調がprofile.mdと一致しているか（食欲優先・直感的・タメ口）
- [ ] ハナの口調がprofile.mdと一致しているか（データで諭す・タメ口・「聞いてないじゃん！」）
- [ ] ハナがデータ・数字に基づいた主張をしているか
- [ ] ケンが食欲・感情で反応しているか
- [ ] 口論シーンが1回以上含まれているか
- [ ] 専門用語に平易な補足説明があるか
- [ ] エンディングが「ハナまとめ → ケンのオチ → ハナの突っ込み」の型になっているか
- [ ] ファイル名が命名規則通りか

### チェック結果の報告フォーマット
---
✅ 通過した項目：
❌ 問題のある項目：
📝 修正内容：
---

### 問題があった場合
修正して再チェックを行う。
全項目✅になってから scripts/ に保存する。
